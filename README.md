# Snake Game in C (Console Version)

A simple console-based Snake game built using C.

## Features
- Control snake using keyboard (W/A/S/D)
- Grow snake when eating food
- Collision detection
- Display score

## How to Run (Windows)
1. Compile:

    gcc snake.c -o snake

2. Run:

    snake.exe
